üìö Search Algorithms

Breadth-First Search (BFS)

Explores all neighbors at the current depth before going deeper. Guarantees shortest path in unweighted graphs.

Depth-First Search (DFS)

Goes as deep as possible down one path before backtracking. Fast but doesn't guarantee shortest path.

Uniform Cost Search (UCS)

Always expands the lowest-cost path first. Finds optimal path in weighted graphs.

A* Search

Uses both actual cost and estimated distance to goal. Most efficient informed search algorithm.

Greedy Best-First

Only considers estimated distance to goal. Fast but not always optimal.

Iterative Deepening DFS

Combines benefits of BFS and DFS. Memory efficient with guaranteed shortest path.

ü§ñ Machine Learning Algorithms

Linear Regression

Finds the best-fit line through data points to predict continuous values. Foundation of supervised learning.

Logistic Regression

Binary classification algorithm using sigmoid function. Outputs probabilities for yes/no decisions.

K-Nearest Neighbors

Instance-based learning that classifies points based on majority vote of K nearest training examples.

Decision Trees

Hierarchical models that split data based on feature values. Easy to interpret and visualize decisions.

Neural Networks

Brain-inspired models with interconnected neurons. Learn complex patterns through backpropagation.

K-Means Clustering

Unsupervised learning that groups similar data points. Iteratively refines cluster centers.

Naive Bayes

Probabilistic classifier based on Bayes' theorem. Assumes feature independence for efficient computation.

‚õ∞Ô∏è Optimization Algorithms

Hill Climbing

Local search that moves to better neighboring states. Fast but can get stuck at local optima.

Simulated Annealing

Probabilistic method that can escape local optima. Temperature controls exploration vs exploitation.

Genetic Algorithms

Evolution-inspired optimization using selection, crossover, and mutation on a population of solutions.

Beam Search

Memory-limited BFS variant that keeps only the K best candidates at each level.

üéÆ Game Playing Algorithms

MinMax Algorithm

Explores game tree to find optimal move. Assumes opponent plays perfectly. Guarantees best outcome.

Alpha-Beta Pruning

Optimizes MinMax by eliminating branches that won't affect final decision. Dramatically reduces search space.

Evaluation Functions

Heuristics to assess non-terminal game states. Enables depth-limited search in complex games.

üß© Constraint Satisfaction

Backtracking Search

Systematically assigns values to variables. Backtracks when constraints are violated.

Arc Consistency

Constraint propagation technique that reduces domain values. Makes search more efficient.

Forward Checking

Eliminates inconsistent values from unassigned variables after each assignment.

Variable Ordering

Heuristics like MRV (Most Constrained Variable) and degree heuristic to choose assignment order.
